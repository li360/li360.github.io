python web 学习——测试安装 Django

[学习笔记] 使用 python 做 web 开发， python web 测试基础学习。

编写测试代码

简单的测试：

  • 引用 selenium
  • browser = webdriver.Firefox() 将打开 Firefox 浏览器，注意这里 Firefox 需要在当前路径变量中，可直接执行，否则会提示找不到 Firefox
  • assert 'Django' in browser.title 看看打开的网页中是否包含 Django

使用 unittest 模块测试

  • 测试方法继承自 unittest.TestCase
  • setUp tearDown 是特殊方法，分别在测试方法之前和之后运行
  • self.assertIn 是 unittest 提供的测试断言的辅助函数

Django 运行

当 Django 开始运行，就可以使用 functional_tests.py 进行测试了
