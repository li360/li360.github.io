问题备忘

windows 命令乱码

例如 ipconfig 命令乱码，可以设置别名，并添加参数
Add | iconv -f GBK -t UTF8 after the CMD command.

没有 telnet 命令

直接寻找 telnet 找不到，需要这个包 inetutils：

启动出现 dll 错误

这个实际上是 Cygwin 的问题

  1. 先把所有 babun 窗口都关闭，不行就重启
  2. 进入 babun 的 cygwin/bin 目录
  3. 以管理员身份运行 ash.exe
  4. 在 ash.exe 的窗口中运行 /usr/bin/rebaseall -v， 然后退出
  5. 再次运行 babun, 完成修复
